As transições CSS permitem que você altere os valores das propriedades sem problemas, durante um determinado período.

Foi necessário uma tag de estilização para manipular o gradiente: <br> <style> </style> <br> Na hora de utilizar o estilo é só utilizar o nome, como por exemplo "grad" <div id="grad"></div> div { width: 100px; -- Propriedades de tamanho height: 100px; -- Propriedades de tamanho background: red; --Cor de fundo transition: width 2s; -- Duração da transição } div:hover { --Enquanto o mouse estiver por cima do objeto width: 600px; }
