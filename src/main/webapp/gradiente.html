Gradientes

Os gradientes CSS permitem exibir transições suaves entre duas ou mais cores especificadas.
CSS define três tipos de gradientes: Gradientes Lineares (baixo/cima/esquerda/direita/diagonal). Para baixo é o modo padrão.
Gradientes radiais (definidos pelo seu centro).
Gradientes cônicos (girados em torno de um ponto central).

Exemplo de Gradientes

Foi necessário uma tag de estilização para manipular o gradiente: <br> <style> #grad2 { -- Aqui é o nome do estilo height: 20px; -- Aqui a altura pré-definida background-color: green; -- Caso o navegador não suporte o gradiente , ele exibirá uma cor única definida aqui background-image: linear-gradient(green, brown); Aqui define a direção e as cores utilizadas } </style> <br> Na hora de utilizar o estilo é só utilizar o nome, como por exemplo "grad" <div id="grad"></div>

Linear


Código:

background-image: linear-gradient(red, yellow);
