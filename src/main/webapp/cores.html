Cores

As cores são especificadas usando nomes de cores predefinidos ou valores RGB, HEX, HSL, RGBA, HSLA.

Exemplo de cores com nomes predefinidos

Código:

<h4 style="background-color:Red;">Red</h4>

Resultado:

Red

Código:

<h4 style="background-color:Blue;">Blue</h4>

Resultado:

Blue

Código:

<h4 style="background-color:Green;">Green</h4>

Resultado:

Green



Exemplo de cores RGB

Código:

<h4 style="background-color:rgb(255, 0, 0);">rgb(255, 0, 0)</h4>


Resultado:

rgb(255, 0, 0)

Código:

<h4 style="background-color:rgb(0, 0, 255);">rgb(0, 0, 255)</h4>


Resultado:

rgb(0, 0, 255)

Código:

<h4 style="background-color:rgb(0, 255, 0);">rgb(0 255, 0)</h4>


Resultado:

rgb(0 255, 0)



Exemplo de cores HEX

Código:

<h4 style="background-color:#ff0000;">#ff0000</h4>


Resultado:

#ff0000

Código:

<h4 style="background-color:#0000ff;">#0000ff</h4>


Resultado:

#0000ff

Código:

<h4 style="background-color:#00ff00;">#00ff00</h4>


Resultado:

#00ff00
